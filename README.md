# Au-Application for job searching
![img](https://imgur.com/6VHT5RK.png)

## Description
The app is about getting news jobs in Bishkek (Kyrgyzstan) published from the hugest platforms of the country.

## Tools used in project
* MVP
* Realm
* Dagger2
* Retrofit
* ButterKnife
